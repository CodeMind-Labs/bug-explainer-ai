# 🤝 Contributing to Bug Explainer AI

Thank you for your interest in contributing! 🚀
We welcome developers of all skill levels.

---

## 📌 Getting Started

1. Fork the repository
2. Clone your fork

```bash
git clone https://github.com/CodeMind-Labs/bug-explainer-ai.git
cd bug-explainer-ai
```

3. Create a new branch

```bash
git checkout -b feature/your-feature-name
```

---

## 🛠️ Making Changes

* Follow the existing project structure
* Write clean and readable code
* Add comments where necessary
* Keep changes small and focused

---

## ✅ Commit Guidelines

Use clear commit messages:

```bash
git commit -m "Add: support for new error type"
git commit -m "Fix: UI alignment issue"
git commit -m "Update: improve error explanation logic"
```

---

## 🚀 Submitting a Pull Request

1. Push your changes

```bash
git push origin feature/your-feature-name
```

2. Open a Pull Request
3. Describe what you changed and why

---
